Motivation

In the game Minesweeper each numbered clue is surrounded by the corresponding number of mines. But have you ever wondered how many possible ways can mines surround each number?

Some years ago I made myself that question, I came up with a table of all possible combinations that are the same under rotation and reflection, and even gave names to them, some using Game of Life lingo (like banana, duoplet or glider,) and others by shape.

It wasn't until recently and under further inspection of the Game of Life algorithms, I realised the section about non-totalistic rules had a table very similar to the one I made, but with a more organized (and useful) naming convention.

So, based on this, I decided to put it all together into an interactive grid, which would automatically identify the pattern and display which non-totalistic rule satisfies, along with the name I gave to the pattern, some of which I modified from the original.

Frequency of each configuration in Minesweeper

Putting the focus back on Minesweeper, it's only natural to ask how often these patterns appear in actual boards.

In order to estimate this I ran some simulations. I am purposefully ignoring nuances like first click or 3BV and simply generating random boards of given size and number of mines, then for each cell the neighborhood configuration is identified.

By default the algorithm does this for every cell, but we can limit the search to non-mines or cells not on the boundary, just for the sake of analysis.

Results

For each board, I generated 10 millions boards. As for the 8s in beginner 9x9, I didn't get any, which is to be expected, as the theoretical probability for an 8 in this size is around 1 in 15 million, compared to the 1 in 2.8 million of beginner 8x8.

This is not an exhaustive search, Javascript is painfully slow and my code is far from being optimized, so the less common numbers 7 and 8 may not robustly reflect the true probability of finding these on boards. Feel free to take a look at the code on this page or its repository if you want to deep further in.
